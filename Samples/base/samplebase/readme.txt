SampleBase

This sample is meant to represent basic functionality in most applications of 
LithTech. It can be used as a starting point for most projects.

Detailed documentation on the C++ code for this project may be
found in the tutorial section of Docs\JupiterHelp.chm

Please use the following commands to move the camera:


Key Configuration:

Up Arrow	- Forwards
Down Arrow	- Backwards
Left Arrow	- Left
Right Arrow	- Right

W		- Forwards
S		- Backwards
A		- Left
D		- Right

Escape		- Quit Sample
