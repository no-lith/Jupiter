Jupiter - Model OBB Sample

This sample shows off the use of Model OBBs (Oriented Bounding Boxes). 

Controls:

W	          : Move forward
A                 : Move left
S                 : Move back
D                 : Move right

Right             : Turn right
Left              : Turn left
Up 	          : Look up
Down              : Look down

Insert            : Move up
Delete            : Move down

1                 : Add row
2                 : Remove row
3                 : Add column
4                 : Remove column
5                 : Remove all
6                 : Previous model
7                 : Next model
B                 : Toggle background color
C                 : Toggle controls UI
T                 : Toggle boxes
