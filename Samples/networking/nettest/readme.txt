Nettest

This sample demonstrates a simple way to create an in-game LAN game browser.

When the sample begins, you will see a menu with a few options on it.

SinglePlayer
 Start a single player game.

Find Game
 Look for active servers.

Host Game
 Start a server.

Once in the game:
Please use the following commands to move the player:


Key Configuration:

Up Arrow	- Forwards
Down Arrow	- Backwards
Left Arrow	- Left
Right Arrow	- Right

W		- Forwards
S		- Backwards
A		- Left
D		- Right

Left Mouse	- Shoot

Escape		- Quit Sample
