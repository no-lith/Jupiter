Bump

This sample demonstrates using DOT3 bump mapping on lightmapped and gouraud surfaces.

Detailed documentation on the C++ code for this project may be
found in the tutorial section of Docs\JupiterHelp.chm


Key Configuration:

Up Arrow	- Forwards
Down Arrow	- Backwards
Left Arrow	- Left
Right Arrow	- Right

W		- Forwards
S		- Backwards
A		- Left
D		- Right

L               - Toggle dynamic light on/off

Escape		- Quit Sample
