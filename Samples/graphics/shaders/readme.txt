Shader Sample

This sample demostrates how to assign a vertex shaders and pixel
shaders to model pieces.

Controls:
W or Up           : Move forward
Left              : Turn left
A                 : Move left
S or Down         : Move back
Right             : Turn right
D                 : Move right
1                 : Add column
2                 : Subtract column
3                 : Add row
4                 : Subtract row
5                 : Subtract all
6                 : Switch to previous model
7                 : Switch to next model
8                 : Switch to previous shader
9                 : Switch to next shader
B                 : Toggle background color
G                 : Toggle controls UI
