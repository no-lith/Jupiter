RenderDemo Sample

This sample shows off the power of the model rendering pipeline.

Controls:

W or Up           : Move forward
Left              : Turn left
A                 : Move left
S or Down         : Move back
Right             : Turn right
D                 : Move right
1                 : Add row
2                 : Remove row
3                 : Add column
4                 : Remove column
5                 : Remove all
6                 : Previous model
7                 : Next model
B                 : Toggle background color
G                 : Toggle controls UI
